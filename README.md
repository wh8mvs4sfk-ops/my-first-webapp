# 📊 个人全栈记账数据看板 (Personal Expense Dashboard)

这是一个基于 Python 构建的轻量级、响应式跨平台记账 Web 应用。项目旨在提供一种极简、直观的日常开销管理方式，并集成了数据可视化与多币种自动汇率换算等企业级业务逻辑。

## ✨ 核心功能 (Features)

*   **📈 可视化数据看板：** 引入 Chart.js，根据消费数据自动生成交互式甜甜圈图，直观展示各项支出比例。
*   **🗓️ 时间维度筛选：** 支持按月份精准查询历史账单，图表与总计金额将根据选定月份动态无感刷新。
*   **💱 多币种与汇率自动折算：** 针对跨国差旅及海淘场景，支持 CNY, JPY, USD, THB 等多种货币录入，并在底层自动按预设汇率折算为统一基础货币（人民币）进行统计分析。
*   **🏷️ 动态分类管理：** 突破固定分类限制，支持用户在前端随时自定义新增消费类目，数据结构动态适配。
*   **📱 响应式设计：** 前端采用 Bootstrap 5 框架，完美适配 Windows 桌面端与移动端手机浏览器的不同屏幕尺寸。

## 🛠️ 技术栈 (Tech Stack)

*   **后端逻辑 (Backend)：** Python 3, Flask (轻量级 Web 框架)
*   **数据存储 (Database)：** SQLite (轻量级本地关系型数据库)
*   **前端展示 (Frontend)：** HTML5, Bootstrap 5, Jinja2 模板引擎
*   **数据可视化 (Data Viz)：** Chart.js
